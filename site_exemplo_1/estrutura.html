HTML: Estrutura da página

HTML significa HyperText Markup Language. Ele organiza o conteúdo da página utilizando diversas tags. 

<h1>Título</h1>
<p>Um parágrafo.</p>
<img src="foto.jpg">

CSS: Aparência da página

CSS significa Cascading Style Sheets. Ele controla cores, fontes, espaçamentos e posicionamento dos elementos. 

body{
    background:#f4f4f4;
}

h1{
    color:steelblue;
}
